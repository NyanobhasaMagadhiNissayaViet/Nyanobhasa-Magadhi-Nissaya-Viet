## Sūtra 33

**Đối chiếu:** caṃ. 5.2.1.27; pā. 6.3.109; mu. 2.34; kā. 2.27.

**Pāḷi:** Tadaminādīni.

**Dịch:** [Các dạng như] "tadamina" v.v. [được xác lập bằng nipātana (quy định đặc biệt)].

## Vutti

**Pāḷi:** Mahāvuttisuttamidaṃ. Tadaminādīni padarūpāni iminā nipātanena sijjhantīti attho.

**Dịch:** Đây là sūtra thuộc mahāvutti (quy tắc tổng quát lớn). Các hình thức từ (padarūpāni) như "tadamina" v.v. (tadaminādīni) được thành tựu (sijjhanti) bằng nipātana (quy định đặc biệt) này, đó là ý nghĩa.

**Pāḷi:** Saralopo byañjane –

**Dịch:** Sara bị lopa [khi đứng] trong [cụm] byañjana, [các dạng bất quy tắc:]

* **lābu** ← alābu [a đầu bị lopa → lābu (bầu/bí)]
* **pidhānaṃ** ← apidhānaṃ [a đầu bị lopa → pidhānaṃ (nắp đậy)]
* **pidahitvā** ← apidahitvā (dvāraṃ pidahitvā) [a đầu bị lopa → pidahitvā (sau khi đóng)]
* **gini** ← aggini [a đầu bị lopa, gg (sarūpa) → g đầu lopa → gini (lửa)]
* **ratnaṃ** ← ratanaṃ [a thứ hai bị lopa → ratnaṃ (châu báu)]
* **nhānaṃ** ← nahānaṃ [a thứ nhất bị lopa → nhānaṃ (tắm rửa)]
* **asnāti** ← asanāti [a thứ hai bị lopa → asnāti (ăn)]
* **hanti** ← hanati [a thứ hai bị lopa → hanti (giết)]
* **hanti kuddho puthujjano** (a. ni. 7.64) [người phàm phu giận dữ thì giết]
* **phalaṃ ve kadaliṃ hanti** (a. ni. 4.68) [quả thật sự giết cây chuối]
* **Sakkāro kāpurisaṃ hanti** [sự cung kính giết kẻ tiểu nhân] [Ledi chú: "katthaci bahuvacanampi dissati", đôi khi dạng số nhiều cũng thấy được]
* **Vikkosamānā tibbāhi, hanti nesaṃ varaṃ varaṃ** (jā. 2.22.2370) [kéo lê bằng [dây] mạnh, giết từng con một trong bọn chúng]

**Pāḷi:** Ivaṇṇalope –

**Dịch:** Trường hợp lopa sara ivaṇṇa (i/ī).

* **cetyāni** ← cetiyāni (ārāmarukkhacetyāni, dha. pa. 188) [i thứ nhất bị lopa → cetyāni (các bảo tháp)]
* **khatyā** ← khattiyā (athatthekasataṃ khatyā, jā. 2.22.594) [i bị lopa, tt (sarūpa) → t đầu bị lopa → khatyā]
* **tithyā** ← titthiyā (tithyā puthuso vadanti, su. ni. 897; Tithyā puthuso niviṭṭhā, su. ni. 898) [i thứ hai bị lopa, tt sarūpa (trong cụm tth) → t đầu bị lopa → tithyā]
* **viddhasto** ← viddhaṃsito [i thứ hai bị lopa, ṃ (niggahīta) bị lopa → viddhasto]
* **utrasto** ← utrāsito [i bị lopa, ā (dīgha) → a → utrasto]
* **Sneho** ← sineho [i bị lopa → sneho (tình thương/dầu)]
* **klesavatthūni** ← kilesavatthūni [i thứ nhất bị lopa → klesa (phiền não)]
* **kriyā** ← kiriyā [i thứ nhất bị lopa → kriyā (hành động)]
* **plavanti** ← pilavanti [i thứ nhất bị lopa → plavanti (trôi nổi)]

**Pāḷi:** Uvaṇṇalope –

**Dịch:** Trường hợp lopa sara uvaṇṇa (u/ū).

* **paddhāni** ← padumāni [u bị lopa → paddhāni (hoa sen)]
* **usmā** ← usumā [u thứ hai bị lopa → usmā (hơi nóng)]

**iccādi (v.v.).**

**Pāḷi:** Saṃyogādibyañjanalopo ca.

**Dịch:** Và lopa byañjana đầu cụm [byañjana, các dạng bất quy tắc:]

* **dukho** ← dukkho (puttānañhi vadho dukho) [k đầu cụm kk (sarūpa) bị lopa → dukho]
* **dukhaṃ** ← dukkhaṃ (mātigho labhate dukhaṃ) [k đầu cụm kk (sarūpa) bị lopa → dukhaṃ]
* **dukhā** ← dukkhā (appassādā kāmā dukhā) [k đầu cụm kk (sarūpa) bị lopa → dukhā]
* **dukhaṃ** ← dukkhaṃ (natthi kāmaparaṃ dukhaṃ, jā. 2.19.118) [k đầu cụm kk (sarūpa) bị lopa → dukhaṃ]
* **sekho** ← sekkho [k đầu cụm kk (sarūpa) bị lopa → sekho]
* **apekhā** ← apekkhā [k đầu cụm kk (sarūpa) bị lopa → apekhā]
* **upasampadāpekho** ← upasampadāpekkho (mahāva. 70) [k đầu cụm kk (sarūpa) bị lopa → upasampadāpekho]

**iccādi (v.v.).**

**Pāḷi:** Sarena saha byañjanalopo.

**Dịch:** Lopa byañjana cùng với sara [đi kèm].

* **paṭisaṅkhā** ← paṭisaṅkhāya (paṭisaṅkhā yoniso, a. ni. 6.58) [ya bị lopa → paṭisaṅkhā]
* **akkhāti** ← akkhāyati [ya bị lopa → akkhāti (nói rõ)]
* **ghāti** ← ghāyati (gandhaṃ ghāti) [ya bị lopa → ghāti (ngửi)]
* **abhiññā** ← abhiññāya [ya bị lopa → abhiññā (sau khi biết rõ)]
* **pariññā** ← pariññāya [ya bị lopa → pariññā (sau khi biết trọn)]
* **adhiṭṭhā** ← adhiṭṭhāya [ya bị lopa → adhiṭṭhā (sau khi quyết định)]
* **patiṭṭhā** ← patiṭṭhāya [ya bị lopa → patiṭṭhā (sau khi thiết lập)]
* **hissa** ← hi + assa (āvīkatā hissa phāsu, mahāva. 134) [a đầu "assa" bị lopa → hissa]
* **assavanatā** ← assavanatāya (assavanatā dhammassa parihāyanti, mahāva. 8) [ya bị lopa → assavanatā]
* **tadārammaṇatā** ← tadārammaṇatāya (vipāko tadārammaṇatā uppajjati, paṭṭhā. 3.1.98) [ya bị lopa → tadārammaṇatā]
* **paramatā** ← paramatāya (dasāhaparamatā dhāretabbaṃ, pārā. 462) [ya bị lopa → paramatā]
* **bhojanatthā** ← bhojanatthāya (nāyaṃ brāhmaṇabhojanatthā, tilodano hehiti, jā. 1.8.1) [ya bị lopa → bhojanatthā]
* **visasenova gārayho, yassatthā rukkharopakā** (jā. 1.3.54) [Ledi chú: "visasenova" = evaṃnāmako rājā eva, tên vị vua ấy vậy]
* **Yassatthā dūramāyanti** (jā. 1.3.28) [Ledi chú: "yassā" = udarassa, "của cái bụng"]
* **atthā** ← atthāya (pitu atthā candavatī, jā. 1.9.66) [ya bị lopa → atthā]
* **upādārūpaṃ** ← upādāya + rūpaṃ [ya bị lopa → upādārūpaṃ]
* **anupādā** ← anupādāya (anupādā vimutto) [ya bị lopa → anupādā (không còn chấp thủ)]
* **saddhāpabbajito** ← saddhāya + pabbajito [ya bị lopa → saddhāpabbajito (xuất gia vì lòng tin)]
* **upanidhāpaññatti** ← upanidhāya + paññatti [ya bị lopa → upanidhāpaññatti]
* **saṃvidhāvahāro** ← saṃvidhāya + avahāro [ya bị lopa → saṃvidhāvahāro]
* **yāti** ← yāyati [ya bị lopa → yāti (đi)]
* **vāti** ← vāyati [ya bị lopa → vāti (thổi)]
* **nibbāti** ← nibbāyati [ya bị lopa → nibbāti (tắt, niết-bàn)]
* **nibbanti** ← nibbāyanti [ya bị lopa → nibbanti (chúng tắt)]
* **pahāti** ← pahāyati [ya bị lopa → pahāti (từ bỏ)]
* **sappatisso** ← sappatissayo [ya bị lopa → sappatisso]
* **suhado** ← suhadayo [ya bị lopa → suhado (người bạn tốt)]
* [Ledi chú: "sabbattha yalopo", tất cả ở đây đều là ya-lopa.]
* **mukharo** ← mukhakharo [kha bị lopa → mukharo]
* **vākkaraṇo** ← vācākaraṇo [samāsa: vācā → vāc → vāk (c ādesa → k); vāk + karaṇo = vākkaraṇo]
* **byappatho** ← vācāpatho [vā → bya (v "byatta" → by, ā → a → bya)] [Ledi chú: "vāssa byattaṃ, rassattañca", vā (của vācā) trở thành byatta, và rassa]
* **byākho** ← viya + kho (evaṃ byākho = evaṃ viya kho) [viya → ya bị lopa → vi → bya (v "byatta" → by, a → ā) → byā + kho = byākho] [Ledi chú: "vissa byattaṃ, dīgho ca yalopo ca", vi trở thành bya (byatta), dīgha, và ya-lopa]

**Pāḷi:** Lolupo, momuho, kukkuco, susukho, roruvo iccādīsu pana atisayatthadīpanatthaṃ padadvittaṃ katvā pubbapadesu akkharalopo.

**Dịch:** Trong các từ như lolupo (tham lam quá mức), momuho (si mê quá mức), kukkuco (hối hận bồn chồn), susukho (an lạc tột bực), roruvo (la hét kinh hoàng) v.v., để nhấn mạnh ý nghĩa cực độ (atisayatthadīpanatthaṃ), từ được nhân đôi (padadvittaṃ katvā), rồi ở từ đứng trước (pubbapadesu) có lopa một số akkhara (akkharalopo).

**Pāḷi:** Padalopo ādimajjhantesu.

**Dịch:** Lopa toàn bộ từ (padalopa), ở đầu (ādi), giữa (majjha), và cuối (anta).

* **datto** ← devadatto [deva bị lopa ở đầu → datto (tên devadatta rút gọn)]
* **assaratho** ← assehi yutto ratho [yutto bị lopa ở giữa → assaratho (xe ngựa)]
* **rūpaṃ** ← rūpabhavo [bhavo bị lopa ở cuối → rūpaṃ (cõi sắc rút gọn)]
* **arūpaṃ** ← arūpabhavo [bhavo bị lopa ở cuối → arūpaṃ (cõi vô sắc rút gọn)]

**iccādi (v.v.).**

**Loparāsi niṭṭhito.**
**Loparāsi hoàn tất.**
