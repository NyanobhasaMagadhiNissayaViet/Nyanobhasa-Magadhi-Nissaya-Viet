## Sūtra 27

**Đối chiếu:** ka. 13; rū. 15; nī. 31.

**Pāḷi:** Paro kvaci.

**Dịch:** [Sara đứng] sau (paro) [bị lopa,] đôi khi (kvaci).

## Vutti

**Pāḷi:** Pubbasaramhā sarūpo vā asarūpo vā paro saro kvaci lopo hoti.

**Dịch:** Từ sara đứng trước (pubbasaramhā), sara đứng sau (paro saro), dù sarūpa (cùng dáng) hay asarūpa (khác dáng), đôi khi (kvaci) bị lopa.

### Sarūpe tāva –

*   **kadāssu** ← kadā + assu ( taṃ kadāssu bhavissati, jā. 2.22.144 ādayo; bản khác đọc taṃ kudassu) [a đầu "assu" bị lopa]
*   **kudāssu** ← kudā + assu (kudāssu nāma dummedho, dukkhassantaṃ karissati, dha. pa. 376) [a đầu "assu" bị lopa]
*   **yadāssa** ← yadā + assa (Yadāssa sīlaṃ paññañca, jā. 2.22.1474) [a đầu "assa" bị lopa]
*   **tadāssu** ← tadā + assu ( tadāssu kaṇhaṃ yuñjanti, jā. 1.1.29) [a đầu "assu" bị lopa] [Ledi chú: kaṇhanti = "kaṇhaṃ" + iti → mahākaṇhagoṇaṃ]
*   **taṇhāssa** ← taṇhā + assa ( taṇhāssa vippahīnā) [a đầu "assa" bị lopa]
*   **māssu** ← mā + assu (māssu kujjha rathesabha) [a đầu "assu" bị lopa]
*   **vāssa** ← vā + assa (satthahārakaṃ vāssa pariyeseyya, pārā. 167, 171) [a đầu "assa" bị lopa]
*   **āgatāttha** ← āgatā + attha (āgatāttha tumhe) [a đầu "attha" bị lopa]
*   **sotukāmāttha** ← sotukāmā + attha (sotukāmāttha tumhe) [a đầu "attha" bị lopa]
*   **māyyo** ← mā + ayyo (māyyo evarūpamakāsi) [a đầu "ayyo" bị lopa]
*   **papaṃ** ← pa + āpaṃ (papaṃ avinduṃ, jā. 1.1.2) [ā đầu "āpaṃ" bị lopa] [Ledi chú: pavaḍḍhaṃ āpaṃ labhiṃsūti attho]
*   **padātave** ← pa + ādātave (nālaṃ kabaḷaṃ padātave, jā. 1.1.27) [ā đầu "ādātave" bị lopa] [Ledi chú: pa+ādātaveti chedo, gaṇhituntattho]
*   **ruppatīti** ← ruppati + iti (ruppatīti rūpaṃ) [i + i → ī, dīgha] [Ledi chú: dīgho]
*   **bujjhatīti** ← bujjhati + iti (bujjhatīti buddho) [i + i → ī, dīgha] [Ledi chú: dīgho]
*   **aggīva** ← aggi + iva (aggīva tappati) [i đầu "iva" bị lopa]
*   **itthīva** ← itthī + iva (itthīva gacchati) [i đầu "iva" bị lopa]
*   **nadīva** ← nadī + iva (nadīva sandati) [i đầu "iva" bị lopa]
*   **mātupaṭṭhānaṃ** ← mātu + upaṭṭhānaṃ (mātupaṭṭhānaṃ) [u đầu "upaṭṭhānaṃ" bị lopa]
*   **pitupaṭṭhānaṃ** ← pitu + upaṭṭhānaṃ (pitupaṭṭhānaṃ) [u đầu "upaṭṭhānaṃ" bị lopa]
*   **yete** ← ye + ete (yete dhammā ādikalyāṇā, cūḷava. 399) [e đầu "ete" bị lopa]

**iccādi (v.v.).**

### Asarūpe –

*   **itissa** ← iti + assa (pāci. 465) [a đầu "assa" bị lopa]
*   **itipi** ← iti + api (pārā. 1) [a đầu "api" bị lopa]
*   **assamaṇīsi** ← assamaṇī + asi (pārā. 135) [a đầu "asi" bị lopa]
*   **akataññūsi** ← akataññū + asi (dha. pa. 383) [a đầu "asi" bị lopa]
*   **vandehaṃ** ← vande + ahaṃ [a đầu "ahaṃ" bị lopa]
*   **sohaṃ** ← so + ahaṃ [a đầu "ahaṃ" bị lopa]
*   **yassadāni** ← yassa + idāni (mahāva. 242) [i đầu "idāni" bị lopa]
*   **chāyāva** ← chāyā + iva (chāyāva anapāyinī, dha. pa. 2) [i đầu "iva" bị lopa]
*   **kathāva** ← kathā + iva (mādisesu kathāva kā) [i đầu "iva" bị lopa]
*   **māva** ← mā + iva (kinnumāva samaṇiyo) [i đầu "iva" bị lopa]
*   **madhuvā** ← madhu + iva (madhuvā maññati bālo, dha. pa. 69) [i đầu "iva" bị lopa; va dīgha thành vā]
*   **cakkhundriyaṃ** ← cakkhu + indriyaṃ (vibha. 219) [i đầu "indriyaṃ" bị lopa]
*   **dveme** ← dve + ime (dveme bhikkhave dhammā, a. ni. 2.3) [i đầu "ime" bị lopa]
*   **tayome** ← tayo + ime (tayome bhikkhave dhammā, a. ni. 3.17) [i đầu "ime" bị lopa]

**iccādi (v.v.).**

**Pāḷi:** Kvacīti kiṃ? Katamā cānanda aniccasaññā (a. ni. 10.60).

**Dịch:** "Đôi khi" (kvaci) nghĩa là gì? [Ví dụ:] "Katamā cānanda aniccasaññā", (a. ni. 10.60). [Ở đây pubbalopa xảy ra thay vì paralopa: a của "ca" bị lopa, ā của "ānanda" giữ → cānanda.]

**Iti paraloparāsi.**
**Đây là nhóm paralopa (lopa sara sau).**
