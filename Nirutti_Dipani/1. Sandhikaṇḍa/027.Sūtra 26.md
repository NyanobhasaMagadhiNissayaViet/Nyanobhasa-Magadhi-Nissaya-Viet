# Sandhividhāna

**Pāḷi:** Atha sandhividhānaṃ dīpiyate.

**Dịch:** Bây giờ, quy tắc nối âm (sandhividhāna) được trình bày.

**Pāḷi:** Lopo, dīgho, rasso, vuddhi, ādeso, āgamo, dvibhāvo, vipallāso.

**Dịch:** [Tám loại sandhī:] lopa (sự cắt bỏ âm), dīgha (sự kéo dài), rassa (sự rút ngắn), vuddhi (sự tăng bậc), ādesa (sự thay thế), āgama (sự thêm âm), dvibhāva (sự nhân đôi), vipallāsa (sự đảo vị trí).

# Loparāsi

## Sūtra 26

**Đối chiếu:** ka. 12; rū. 13; nī. 30.

**Pāḷi:** Saro lopo sare.

**Dịch:** Sara bị lopa (cắt bỏ) khi có sara (sare) [đứng sau].

### Nirutti

**Pāḷi:** Luppatīti lopo.

**Dịch:** "Cái bị cắt bỏ (luppati)", vì vậy gọi là lopo.

### Vutti

**Pāḷi:** Sare pare sarūpo vā asarūpo vā pubbo saro lopo hoti.

**Dịch:** Khi có sara đứng sau (sare pare), sara đứng trước (pubbo saro), dù sarūpa (cùng dáng) hay asarūpa (khác dáng), bị lopa.

#### Sarūpe tāva –

**Avaṇṇe:**
*   **lokaggo** ← loka + aggo (apa. thera 1.12.57) [a cuối "loka" bị lopa]
*   **bhavāsavo** ← bhava + āsavo (dha. sa. 1102) [a cuối "bhava" bị lopa]
*   **avijjāsavo** ← avijjā + āsavo (dha. sa. 1102) [ā cuối "avijjā" bị lopa]
*   **avijjānusayo** ← avijjā + anusayo (vibha. 949) [ā cuối "avijjā" bị lopa]

**Ivaṇṇe:**
*   **munindo** ← muni + indo [i cuối "muni" bị lopa]
*   **munīrito** ← munī + īrito [ī cuối "munī" bị lopa]
*   **varavādīrito** ← varavādī + īrito [ī cuối "varavādī" bị lopa]
*   **itthindriyaṃ** ← itthi + indriyaṃ (vibha. 219) [i cuối "itthi" bị lopa]

**Uvaṇṇe:**
*   **bahūpakāro** ← bahu + upakāro (jā. 1.22.588) [u cuối "bahu" bị lopa]
*   **bahūmi** ← bahukā + ūmi [ā cuối "bahukā" bị lopa]
*   **sarabhūmi** ← sarabhuyā + ūmi [ā cuối "sarabhuyā" bị lopa]
*   **sarabhūdakaṃ** ← sarabhuyā + udakaṃ [ā cuối "sarabhuyā" bị lopa]

#### Asarūpe –

*   **sotindriyaṃ** ← sota + indriyaṃ (vibha. 219) [a cuối "sota" bị lopa]
*   **kāmupādānaṃ** ← kāma + upādānaṃ [a cuối "kāma" bị lopa]
*   **bhavesanā** ← bhava + esanā (dī. ni. 3.305) [a cuối "bhava" bị lopa]
*   **bhavogho** ← bhava + ogho (dha. sa. 1156) [a cuối "bhava" bị lopa]
*   **tuṇhassa** ← tuṇhī + assa (so tuṇhassa, pārā. 381) [ī cuối "tuṇhī" bị lopa]
*   **diṭṭhānusayo** ← diṭṭhi + anusayo (vibha. 949) [i cuối "diṭṭhi" bị lopa]
*   **diṭṭhupādānaṃ** ← diṭṭhi + upādānaṃ [i cuối "diṭṭhi" bị lopa]
*   **diṭṭhekaṭṭhaṃ** ← diṭṭhi + ekaṭṭhaṃ [i cuối "diṭṭhi" bị lopa]
*   **diṭṭhogho** ← diṭṭhi + ogho (dha. sa. 1156) [i cuối "diṭṭhi" bị lopa]
*   **mudindriyaṃ** ← mudu + indriyaṃ (mahāva. 9) [u cuối "mudu" bị lopa]
*   **puttā matthi** ← puttā + me + atthi (dha. pa. 62) [e cuối "me" bị lopa]
*   **urassa** ← ura + assa (urassa dukkho, pāci. 402) [a cuối "ura" bị lopa]
*   **asantettha na dissanti** ← asante + ettha + na + dissanti (dha. pa. 304) [e cuối "asante" bị lopa]

**iccādi (v.v.).**

**Iti pubbaloparāsi.**
**Đây là nhóm pubbalopa (lopa sara trước).**
