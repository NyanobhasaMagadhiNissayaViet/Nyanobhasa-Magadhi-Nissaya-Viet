## Rassarāsi

**Pāḷi:** Rassasandhimhi gāthāvasena tāva –

**Dịch:** Trong sandhi rassa, trường hợp vì vần luật trước.

* **yiṭṭhaṃva** ← yiṭṭhaṃ + vā (yiṭṭhaṃva hutaṃva loke, dha. pa. 108) [ā của "vā" → a (rassa) vì gāthā → va]
* **bhovādi** ← bhovādī (bhovādi nāma so roti, dha. pa. 396; su. ni. 625) [ī của "bhovādī" → i (rassa) vì gāthā]
* **yathābhāvi** ← yathābhāvī (yathābhāvi guṇena so) [ī của "yathābhāvī" → i (rassa) vì gāthā]

**iccādi (v.v.).**

**Pāḷi:** Āgame ya, ra, dāgamesu pubbarasso –

**Dịch:** Trong các trường hợp āgama ya, ra, da, sara trước [āgama] bị rút ngắn (pubbarasso).

* **yathayidaṃ** ← yathā + idaṃ (yathayidaṃ, a. ni. 1.1-4) [y (āgama) xen vào; ā trước y → a (rassa)]
* **tathayidaṃ** ← tathā + idaṃ [y (āgama) xen vào; ā trước y → a (rassa)]
* **yathariva** ← yathā + iva (yathariva amhākaṃ bhagavā) [r (āgama) xen vào; ā trước r → a (rassa)]
* **tathariva** ← tathā + iva (tathariva bhikkhusaṅgho) [r (āgama) xen vào; ā trước r → a (rassa)]
* **sammadeva** ← sammā + eva (sammadeva samācare, saṃ. ni. 1.112) [d (āgama) xen vào; ā trước d → a (rassa)]
* **sammadakkhāto** ← sammā + akkhāto (sammadakkhāto mayā satisambojjhaṅgo, saṃ. ni. 5.194) [d (āgama) xen vào; ā trước d → a (rassa)]

**iccādi (v.v.).**

**Pāḷi:** Saṃyogarasso nāma bahulaṃ labbhati.

**Dịch:** Rassa trước cụm byañjana (saṃyogarassa) được tìm thấy rất thường xuyên (bahulaṃ).

* **akkamo** ← ā + kamo [ā → a (rassa) trước cụm kk]
* **parakkamo** ← parā + kamo [ā cuối "parā" → a (rassa) trước cụm kk]
* **akkhāto** ← ā + khāto [ā → a (rassa) trước cụm kkh]
* **aññā** ← ā + ñā [ā → a (rassa) trước cụm ññ]
* **aññindriyaṃ** ← aññā + indriyaṃ [ā cuối "aññā" → a (rassa)]
* **aññātaṃ** ← aññā + ataṃ (aññātaṃ bhagavā) [ā cuối "aññā" → a (rassa)]
* **aññātaṃ** ← aññā + ataṃ (aññātaṃ sugata) [ā cuối "aññā" → a (rassa)]
* **attharaṇaṃ** ← ā + tharaṇaṃ [ā → a (rassa) trước cụm tth]
* **apphoṭeti** ← ā + phoṭeti [ā → a (rassa) trước cụm pph]
* **alliyati** ← ā + liyati [ā → a (rassa) trước cụm ll]
* **acchindati** ← ā + chindati [ā → a (rassa) trước cụm cch]
* **assādo** ← ā + sādo [ā → a (rassa) trước cụm ss]
* **ābhassaro** ← ābhā + ssaro [ā cuối "ābhā" → a (rassa) trước cụm ss]
* **pabhassaro** ← pabhā + ssaro [ā cuối "pabhā" → a (rassa) trước cụm ss]
* **sabbaññutaññāṇaṃ** ← sabbaññū + ta + ñāṇaṃ [ū cuối "sabbaññū" → u (rassa)]
* **lābhimhi** ← lābhī + mhi (jhānassa lābhimhi vasimhi, pārā. 203-204) [ī cuối "lābhī" → i (rassa) trước cụm mh]
* **vasimhi** ← vāsī + mhi [ī cuối "vāsī" → i (rassa); ā đầu → a (rassa)]

**Pāḷi:** Tā, topaccayesupi rasso –

**Dịch:** Trước paccaya -tā và -to, cũng [xảy ra] rassa.

* **kataññutā** ← kataññū + tā [ū cuối "kataññū" → u (rassa) trước -tā]
* **atthaññutā** ← atthaññū + tā [ū cuối "atthaññū" → u (rassa) trước -tā]
* **dhammaññutā** ← dhammaññū + tā [ū cuối "dhammaññū" → u (rassa) trước -tā]
* **kaññato** ← kaññā + to [ā cuối "kaññā" → a (rassa) trước -to]
* **nadito** ← nadī + to [ī cuối "nadī" → i (rassa) trước -to]
* **vadhuto** ← vadhū + to [ū cuối "vadhū" → u (rassa) trước -to]

**Pāḷi:** Samāse –

**Dịch:** Trong samāsa (hợp từ).

* **itthipumaṃ** ← itthī + pumaṃ [ī cuối "itthī" → i (rassa) trong samāsa]
* **itthiliṅgaṃ** ← itthī + liṅgaṃ [ī cuối "itthī" → i (rassa) trong samāsa]
* **itthibhāvo** ← itthī + bhāvo [ī cuối "itthī" → i (rassa) trong samāsa]
* **sabbaññubuddho** ← sabbaññū + buddho [ū cuối "sabbaññū" → u (rassa) trong samāsa]

**iccādi (v.v.).**

**Iti rassarāsi.**
**Đây là nhóm rassa [hoàn tất].**

**Dīgha rassarāsi niṭṭhito.**
**Dīgha-Rassarāsi hoàn tất.**
